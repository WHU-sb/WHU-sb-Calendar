Guide

导入教程

按设备选择导入或订阅方式。

Windows
  1. 下载 .ics 文件
  2. 右键选择打开方式
  3. 选择日历应用导入
iOS
  1. 设置 -> 日历 -> 账户
  2. 添加账户 -> 其他
  3. 添加已订阅的日历
Apple 官方教程
macOS
  1. 下载 .ics 文件
  2. 双击文件
  3. 系统日历中确认导入
Outlook Web
  1. 打开 outlook.live.com/calendar
  2. 添加日历 -> 从 Web 订阅
  3. 输入订阅链接
打开 Outlook
Google Calendar
  1. 打开 URL 导入页面
  2. 粘贴订阅链接
  3. 确认添加并等待同步
打开 Google 导入页
